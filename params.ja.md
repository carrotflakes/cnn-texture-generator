# channels

ネットワークの中間パラメータのチャネル数です。

小さいと単純な模様、少ない色数になる傾向があります。
大きすぎると生成に時間がかかります。
2～4 がおすすめです。
network seed よりも先に決めるべきパラメータです。

# network seed

ネットワークのパラメータを決定する乱数シードです。

この値がテクスチャの形状や色を決定します。

# fineness

CNN 層を何回掛けるかを決定するパラメータです。

大きくするほど、カメラを引いたように細かい模様が出ます。
小さい値で確認することをおすすめします。
テクスチャをタイリングして使う場合は、値を上げると繰り返しが目立たなくなります。

# inject

ノイズの注入量を表すパラメータです。

小さい値にするとのっぺりとしたテクスチャになります。大きい値にすると細かい模様が出ます。
1 が標準値です。

# output_size

出力画像のサイズです。

解像度のみならず、模様も変化することがあります。
大きいほど生成に時間がかかります。

# noise seed

ノイズの乱数シードです。

テスクチャのテイストを変えずに違う模様を生成したい場合に使います。
